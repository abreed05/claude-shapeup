# shapeup:retro — Planned

This skill is a reserved slot in the `shapeup` plugin family.

**Planned purpose:** facilitate a post-cycle retrospective that separates **shaping** lessons (what did we mis-shape?) from **building** lessons (what did we hit during the build?) from **betting** lessons (what should we have bet differently?). Produces a retro artifact that feeds back into the next cycle.

**Status:** not yet implemented.

For now:
- Use `shapeup:shape` to shape pitches with clearer rabbit holes and no-gos so retros have something to compare against.
- See the plugin README for the family roadmap.
